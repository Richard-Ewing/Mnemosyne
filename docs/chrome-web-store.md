## Detailed Description
Mnemosyne adds a private semantic ledger to your AI tools so meaning, context, and important facts don't disappear between conversations.

Instead of storing chat history, Mnemosyne records governed meaning with provenance and auditability. This allows AI systems to reference prior context, surface contradictions, and reduce hallucinations caused by silent context loss.

### Key features:

- Persistent context across AI tools and models
- Visibility into what an AI has previously accepted as true
- Automatic contradiction detection
- User-controlled reinforcement and decay
- No training on your data
- Explicit, user-initiated capture only

Mnemosyne is not an AI assistant.
It does not read pages automatically.
It only records what you explicitly choose.

## Data Usage Answers

### Does the extension collect personal data?
Yes, but only user-selected content.

### What data is collected?
- User-selected text upon explicit action
- Optional source metadata (URL)
- Account identifier for authentication

### What data is NOT collected?
- Browsing history
- Keystrokes
- Background page content
- Personal data without user action

### How is data used?
Data is used exclusively to provide semantic ledger functionality, including meaning representation and contradiction detection.

### Is data sold or shared?
No. Data is not sold and is not shared for advertising or analytics.

### Is data encrypted?
Yes. All data is encrypted in transit and at rest.

### Can users delete their data?
Yes. Users can export or permanently delete their data at any time.
